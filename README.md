# Project Setup Instructions

## Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd RAVID
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the frontend application:
   ```bash
   npm run start
   ```
